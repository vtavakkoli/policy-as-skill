# AI Pilot Policy
version: v2

## pilot-policy-v2
The current pilot policy requires a named responsible owner, policy-version logging, security assessment, data-protection review, rollback criteria, and review before any external generative AI pilot can proceed. Pilots without a named owner are not allowed.

## policy-update
When policy versions conflict, the current approved version applies. Decisions must cite the policy version used and record the policy hash to support later audit reconstruction.
